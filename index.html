📊 Quick Reference Card
🎯 Signal Scores
DLI = 1 - (t/t_max)
CMS = 1 - weight(error)
ADRC = weighted mean(correct_rate)
CED = (attempts × time) clamped [0.5,1]
LVC = (CMS_prev - CMS_curr)/count
⚖️ BWV Formula
BWV = (w₁×DLI) × (w₂×CMS) × (w₃×ADRC) × (w₄×CED) × (1 + w₅×LVC)
w₁=0.20 w₂=0.30 w₃=0.20 w₄=0.30 w₅=0.25
🧮 IRT Updates
θ_new = θ + 0.15 × (actual×BWV - P_i)
χ_new = χ - 8 × (actual - P_i) × ADRC
P_i = 1/(1 + exp(-s_eff×(θ-χ)))
⭐ Glicko Updates
Mt_new = Mt + K × mean(actual-E_i) × mean(BWV)
K = 24 × (1 + 0.25×LVC)
φ²_new = min(3, 1/(1/φ_hat² + I))

📡 Signal Scores (Detailed)
DLI_score
DLI = 1 - (t_response / t_max)
Range: 0.0 to 1.0 • Speed of response
CMS_score
CMS = 1 - CMS_weight[error_category]
Range: 0.0 to 1.0 • Error type severity
ADRC_score
ADRC = 0.2×R_easy + 0.4×R_medium + 0.4×R_hard
Performance across difficulty tiers
CED_score
CED = (non_guess/total) × (avg_time_ratio)
Clamped [0.5, 1.0] • Engagement level
LVC_score
LVC = (CMS_prev - CMS_curr) / quiz_count
Range: -1.0 to +1.0 • Improvement rate
⚖️ Behavioral Weight Vector (BWV)
BWV = (w₁×DLI) × (w₂×CMS) × (w₃×ADRC) × (w₄×CED) × (1 + w₅×LVC)
Weight Value Component Why This Weight
w₁0.20DLISpeed matters but not everything
w₂0.30CMSError type is highly informative
w₃0.20ADRCConsistency across difficulties
w₄0.30CEDEffort and engagement matter
w₅0.25LVCImprovement gets bonus (amplifier)
📈 IRT & Glicko Core Equations
IRT θ Update
s_i_eff = s_i × (1 - 0.4 × CMS_weight)
P_i = 1 / (1 + exp(-s_i_eff × (θ - χ_i)))
Q_i = actual_i × BWV
θ_new = θ + 0.15 × (Q_i - P_i)
χ_new = χ - 8 × (actual_i - P_i) × ADRC
Glicko G* Update
φ_hat² = φ² + (Δt × Vt²) [clamped to 3.0]
g_φ = 1 / √(1 + 3φ_hat²/π²)
E_i = 1 / (1 + exp(-g_φ × (Mt - χ_i)))
K = 24 × (1 + 0.25 × LVC)
Mt_new = Mt + K × mean(actual_i - E_i) × mean(BWV)
φ²_new = min(3, 1/(1/φ_hat² + I))
Vt_new = Vt × (1 + 0.1×CMS_weight - 0.05×LVC)
🔄 Mode Comparison: Tournament vs Matchmaking
🏆 TOURNAMENT MODE
Updates: G*, θ*, AppScore (permanent)
Constants: α=0.15, K_base=24
Output: AppScore (0-100)
Purpose: High-stakes assessment
Formula: G_norm^λ × θ_norm^(1-λ) × CED × (1 + LVC×0.15)
🤝 MATCHMAKING MODE
Updates: Only H-Score (G/θ discarded)
Constants: α_s=0.195, K_s=36
Output: H-Score (0-100)
Purpose: Practice & matchmaking
Formula: (0.35CCR+0.25DPI+0.2SES+0.2CI)×(1+LVC×0.2)×100
