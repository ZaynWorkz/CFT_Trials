Overview

CTF (Cognitive Trajectory Fusion) combines behavioral signals to compute performance scores.

DLI

Decision Latency Index

CMS

Cognitive Mistake Signature

CED

Engagement Density

Signals
DLI = 1 - (t / t_max)
CMS = 1 - weight(error)
CED = (engagement × time)
Behavioral Weight Vector
BWV = (0.2×DLI)(0.3×CMS)(0.2×ADRC)(0.3×CED)(1 + 0.25×LVC)
